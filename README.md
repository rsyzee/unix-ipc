# unix-ipc
is a Interprocess Communication (IPC) using local unix socket. used to communicate and sharing data between processes
# Installation & run
```sh
git clone https://github.com/ryszee/unix-ipc.git
cd unix-ipc
mkdir build
cd build
cmake ..
make -j4 
```

run ipc server :
```sh
./ipc-server
```
run ipc client :
```sh
./ipc-server
```
